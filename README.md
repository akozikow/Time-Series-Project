# Time-Series-Project
For this project, I plan to try to forecast the S&P 500. In addition to the publicly available performance of the index itself, I will be using a transformer LLM (e.g., RoBERTa) to do sentiment analysis on headlines from major financial news outlets (likely obtained via scraping) to see if I can gauge a bullish or bearish sentiment from them. I will save the S&P 500 performance and the overall sentiment for that time period in an SQL server to acculate my own data to make predictions with. 

The intention of this project is to gain experience in a number of important skills (Time Series Forecasting, SQL, API/Web Scraping, Sentimetn Analysis) and use them to create a product of value.
